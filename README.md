## Selected cases

### AI-генератор VK Shorts для городской администрации

Telegram-бот для полностью автоматизированной генерации новостных
видеороликов с персонажем-аватаром.\
**Value:** повышение качества контента, экономия времени сотрудников,
рост просмотров и ER.

### Wildberries autoreply bot

AI-бот для автоматических ответов на отзывы покупателей с использованием
ChatGPT.\
**Value:** сокращение времени работы менеджера с 10+ часов до минут в
месяц, отсутствие неотвеченных отзывов, рост CR.
